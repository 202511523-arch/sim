# All-in-One Chemistry Assistant

An integrated chemistry tool web application for chemistry students.

## Key Features

1. **Smart Calculator** - Engineering Calculator + Chemical Equilibrium Solver
2. **Molecular Editor** - SMILES-based Molecular Structure Editor
3. **Data Analyzer** - Data Analysis and Regression Tool
4. **3D Visualizer** - Protein/Molecule 3D Visualization
5. **Chemical Search** - PubChem Compound Search

## Installation

```bash
pip install -r requirements.txt
```

## How to Run

```bash
streamlit run app.py
```

## Tech Stack

- **Backend:** Python 3.8+
- **Framework:** Streamlit
- **Chemistry Libraries:** RDKit, PubChemPy
- **Computation:** SymPy, NumPy, Pandas
- **Visualization:** Plotly, py3Dmol
